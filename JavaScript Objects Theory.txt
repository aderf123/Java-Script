An object in JavaScript is a data structure used to store multiple related values in one place. These values are stored as key-value pairs. Example: 

let student = {
  name: "Freda",
  age: 25,
  course: "Web Development"
};


Difference Between Objects and Variables

A variable stores a single value:
let name = "Freda";
An object stores multiple related values:
let student = {
  name: "Freda",
  age: 25
};


Why We Use Objects
Objects are used to:
	•	Organize related data
	•	Make code cleaner and easier to read
	•	Represent real-world things (like a student, bank account, user, etc.)
	•	Allow both data and functions to exist together


1. Property
A property is a piece of data stored inside an object.
Example:  name: "Freda"

Method
A method is a function inside an object.
Example:
introduce: function() {
  console.log("Hello");
}

3. Difference Between Property and Method
	•	Property → stores data
	•	Method → performs an action
Example:
let person = {
  name: "Freda",        // property
  greet: function() {   // method
    console.log("Hi");
  }
};


4. Difference Between Function and Method

A function is a standalone block of code:
function greet() {
  console.log("Hello");
}

A method is a function inside an object:
let obj = {
  greet: function() {
    console.log("Hello");
  }
};

Accessing Object Values

There are two ways to access object properties:
	1.	Dot Notation
student.name

	2.	Bracket Notation
student["name"]

The this Keyword

The this keyword refers to the current object.
Example:
let student = {
  name: "Freda",
  introduce: function() {
    console.log(this.name);
  }
};


Nested Objects

An object can contain another object inside it.
Example:
let student = {
  name: "Freda",
  address: {
    city: "Lagos",
    country: "Nigeria"
  }
};
